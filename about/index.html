关于

我是 Charles，一名 Web 全栈开发者、工具库作者。
高中在读，成绩上游，初二接触编程，初三入门 Web。

我追求极致工程化。不论是变量名、换行空行、提交记录还是项目架构，我都会充分斟酌直到自己真正满意；对于杂乱的实现，我会果断重构，将代码的质量和可读性提升至我能达到的最高水平。一年中，从学着写 HTML、CSS、JavaScript 的小萌新，到能够独立完成前端后端、造轮子填补生态空白、为开源工程贡献代码的开发者，我对我的进步相当满意。

我自以为对 TypeScript（JavaScript）造诣颇高。对于一个函数的 path 参数，我宁愿为其写几百行的 Path<...> 类型来实现编辑器的自动补全，也绝不会像大多数开发者那般扔下一个 string 类型便一走了之。
其次则是 Python。同样，我对其类型也有着很高的要求。纵使 Python 的类型系统、类型生态当前仍相当不成熟，我也会尽力将类型注解到最详细，而非像大多数开发者那般根本不注释类型亦或是一堆 Any。

  • 最喜欢的后端框架是 Django + Django REST Framework：带我第一次走进了 Web 的世界；其极致组件化的架构、庞大繁荣的生态始终令我叹为观止；其 ORM 的设计是我见过最优秀的。
  • 最熟悉的后端框架是 NestJS：带我第一次认识了依赖注入、开发过程中极大地加深了我对 TypeScript 的理解
  • 最熟悉的前端框架是 Vue：带我第一次了解了前后端分离的概念、带我第一次走进 NodeJS 的世界。
